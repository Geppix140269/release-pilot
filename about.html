What We Do

ReleasePilot automates the boring parts of software releases:

  • AI-generated release notes from commit messages
  • Automatic semantic versioning
  • AI code review for Azure DevOps
  • Deployment triggers and notifications

What We Don't Do

  • Replace human developers
  • Guarantee perfect AI output
  • Provide 24/7 support (we sleep)
  • Custom development work

Why We Built This

Simple: We hate writing release notes.

Every sprint, same story:

  • Collect all commits
  • Try to remember what each one does
  • Write user-friendly descriptions
  • Format everything nicely
  • Post to GitHub/Azure DevOps

Total waste of 2-3 hours per month. So we automated it.
