Quick Setup Guide

Get started in 3 simple steps

1 Get Your OpenAI API Key

  1. Sign up at platform.openai.com (get $5 free credit)
  2. Go to API Keys ‚Üí Create new secret key
  3. Copy your key (starts with sk-...)

2 Add to GitHub Secrets

In your GitHub repository: 
Settings ‚Üí Secrets ‚Üí Actions ‚Üí New repository secret
Name: OPENAI_API_KEY
Value: [Your OpenAI API key]

3 Add ReleasePilot Action

Create .github/workflows/release.yml: 
name: ReleasePilot
on:
  push:
    branches: [main]
  pull_request:

jobs:
  release:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: releasepilot/action@v1
        with:
          openai-api-key: ${{ secrets.OPENAI_API_KEY }}

That's it! ReleasePilot will now handle your releases automatically.
